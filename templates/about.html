About ML Project

ML Project is a machine learning project aimed at predicting house prices using various techniques and algorithms. Our goal is to develop accurate models that provide valuable insights into the real estate market.

The project utilizes machine learning algorithms such as Linear Regression, Lasso, and techniques like Z-score for outlier handling. These methodologies enable us to build robust models for accurate price predictions.
